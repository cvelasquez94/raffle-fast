# Raffle Fast FE

A raffle management application built with React, TypeScript, Vite, and Supabase.

## Project Description

This application allows users to create raffles, manage raffle numbers, track purchases, and administer raffle events.

## Tech Stack

This project is built with:

- React 18
- TypeScript
- Vite
- Supabase
- shadcn/ui
- Tailwind CSS

## Getting Started

### Prerequisites

You need to have Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd raffle-fast-fe

# Install the necessary dependencies
npm i

# Start the development server
npm run dev
```

## Development

The development server runs on port 8080. Visit `http://localhost:8080` after starting the dev server.
